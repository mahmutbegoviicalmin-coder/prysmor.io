Prysmor Panel v2.1.0

This installer sets up the Prysmor AI VFX extension for Adobe Premiere Pro — including the Identity Lock engine that preserves your subject's face across AI-generated effects.

The installer will automatically:

  • Install the Prysmor CEP panel into Premiere Pro
  • Install the Identity Lock engine (face_embedding_server.py)
  • Install Python AI dependencies (first install only, ~5–10 min in background)
  • Create a login item so Identity Lock starts automatically on every login
  • Clear CEP caches so the panel loads immediately after restart

Click Install to continue. Your Mac password will be required.

Requires macOS 10.15 or later · Adobe Premiere Pro 2020–2025 · Python 3 (pre-installed on most Macs)
